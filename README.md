# rust-async
Asynchronous Rust

## To run main

### Main
```rust
cargo run 
```

## To run examples

### Blocking
```rust
cargo run --example blocking
```

### Non-Blocking
```rust
cargo run --example non-blocking
```

### Non-Blocking Single Thread
```rust
cargo run --example non-blocking-single
```

### Counter with thread
```rust
cargo run --example counter-thread
```

### Counter with tokio
```rust
cargo run --example counter-tokio
```
